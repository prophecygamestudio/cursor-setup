# Cursor Setup

One-line install for Windows (works from CMD or PowerShell):

**From CMD:**
```cmd
powershell -ExecutionPolicy Bypass -Command "iex (irm 'https://raw.githubusercontent.com/RallyHereInteractive/cursor-setup/main/install.ps1')"
```

**From PowerShell:**
```powershell
iex (irm 'https://raw.githubusercontent.com/RallyHereInteractive/cursor-setup/main/install.ps1')
```

**Or simply download and run `install.ps1` from this repository.**
